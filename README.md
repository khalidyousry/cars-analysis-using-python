📌 Project Overview
This project performs a comprehensive exploratory data analysis (EDA) on a classic automotive dataset containing 406 vehicle records from 1970 to 1982. The goal is to analyze fuel efficiency trends, engine performance across different regions (US, Japan, Europe), and categorize vehicles based on their physical and mechanical traits.

🛠️ Tech Stack
Language: Python 3.x

Libraries: * Pandas (Data Manipulation)

Matplotlib & Seaborn (Data Visualization)

NumPy (Mathematical operations)

🔍 Key Questions Answered
Regional Efficiency: Which region produced the most fuel-efficient cars? (Spoiler: Japan leads with an average of 30.45 MPG).

Performance Trends: How has fuel economy (MPG) evolved over a decade of manufacturing?

Engineering Correlation: What is the relationship between Horsepower, Weight, and Acceleration?

Custom Classification: Developed a logic to categorize cars into:

Racing Class: High power-to-weight ratio.

Family Class: High fuel efficiency (MPG).

Standard Class: Balanced daily drivers.

📊 Visualizations & Insights
The project includes a comprehensive dashboard covering:

MPG Trends: Visualizing the steady increase in fuel efficiency over time.

Correlation Heatmap: Analyzing how weight negatively impacts MPG.
